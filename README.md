#  Counter Application

#### In contrast to DART, this is simple Flutter counter application using PYTHON.

## Packages
1. [flet](https://flet.dev/) : The fastest way to build flutter apps using Python

## Usage
```bash
pip3 install flet
```

## Run on Native OS
```bash
python3 main.py
```

## Run on WEB
```bash
flet.app(target=main, view=flet.WEB_BROWSER)
```
